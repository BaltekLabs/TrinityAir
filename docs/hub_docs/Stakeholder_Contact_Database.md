## PRIORITY CODING SYSTEM

### Priority Levels
- **A1 - Critical:** Immediate contact (next 2 weeks)
- **A2 - High:** Contact within 30 days
- **B1 - Important:** Contact within 60 days
- **B2 - Strategic:** Contact within 90 days
- **C - Future:** Contact when appropriate

### Engagement Types
- **IM - Initial Meeting:** Executive summary presentation
- **PP - Partnership Proposal:** Detailed partnership discussion
- **FP - Funding Proposal:** Investment presentation
- **GM - Government Meeting:** Municipal/regulatory discussion
- **TM - Technical Meeting:** Technology partnership

2. One-sentence project description
3. Key benefit relevant to recipient
4. Request for brief meeting/call
5. Executive summary attachment
6. Professional signature with contact information

- [ ] Executive summary printed and ready

- [ ] Background research on contact/organization

- [ ] Calendar availability confirmed
